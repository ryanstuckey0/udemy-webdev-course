Columns and Bootstrap Breakpoints

Bootstrap has breakpoints that can be used to control behavior across different screen sizes.

More info can be seen here: https://getbootstrap.com/docs/5.3/layout/breakpoints/#available-breakpoints

For example, observe how the below column behaves with the following bootstrap classes: col-sm-1 col-med-6 col-lg-12

col

In the above, once the screen shrinks below 576px (sm breakpoint), then notice how it takes up all 12 slots since the sm breakpoint only applies when the screen is at least 576px wide


In the below, when we have multiple columns, they will default to 100% once the screen shrinks past their specified breakpoint

Having a screenwider than the md breakpoint will cause them to take up 3 and 6 columns, respectively. However, once shrinking past the md breakpoint, the columns will switch to take up 100% as if they just had the col class on them.
